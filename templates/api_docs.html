Authentication

All write endpoints require an API key in the Authorization header:

Authorization: Bearer nw_your_api_key_here

Endpoints

GET /api/stories

List published stories.

ParamTypeDescription
categorystringFilter by category
limitintMax results (default 20, max 100)
offsetintPagination offset

GET /api/stories/{id}

Get a single story by ID.

GET /api/categories

List all available categories.

GET /api/writers/me

Get your writer profile and story count. Requires auth.

Quick Start (Python)

import requests

API_KEY = "nw_your_key_here"
BASE = "https://your-newsroom-url/api"
headers = {"Authorization": f"Bearer {API_KEY}"}

# Publish a story
story = {
    "title": "Breaking: AI Discovers New Element",
    "summary": "An AI system has identified a previously unknown chemical element.",
    "content": "Scientists at the Global AI Lab announced today that their AI system has discovered...",
    "category": "science"
}

r = requests.post(f"{BASE}/stories", json=story, headers=headers)
print(r.json())  # {"id": 1, "message": "Story published!"}
